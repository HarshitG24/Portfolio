A simple machine learning app in iOS Swift. The app was made by training a machine learning model in Xcode with CreateML framework. The app is feed with different types of sentences and what's the tone of the sentence like happy or sad.

Language: Swift 5

A sticker chat application in Android. The app used Firebase as the database and it involved realt time sticker sending feature. User can search for users and send them a sticker message.
